# Microbialplus_reservoir_microbiome
R script of the analysis used in Liu et al. 'Bio-chemical Complex Flooding Triggers Synergistic Oil Displacement by Reshaping the Reservoir Microbiome'.

Other data used in the analysis can be found in the Supplementary Information of the paper.

Raw read sequences of this study have been deposited in the NCBI Sequence Read Archive (RSA) under the BioProject accession number PRJNA1208403.
